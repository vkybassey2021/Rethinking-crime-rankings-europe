# Rethinking-crime-rankings-europe
Data visualization project analyzing EU crime patterns using ranking, per-capita, trend, and profiling approaches with Eurostat data.
# Europe Crime Analysis

Analysis of Eurostat crime data (2014–2023) exploring how rankings change when adjusted for population, how crime evolves over time, and how countries can be profiled based on risk and trend.

---

## 📂 Data Source

- Eurostat Crime Statistics (2014–2023)  
- Eurostat Population Data  
  

---

## ⚙️ Methodology

- Cleaned and standardised crime and population datasets  
- Filtered to EU-27 countries  
- Calculated:
  - Raw crime rankings  
  - Crime per 100,000 people (per capita)  
  - Percentage change in crime over time  
- Combined metrics to profile countries based on:
  - Risk per person  
  - Trend direction (increasing or decreasing)  

---

## 📈 Outputs

The project includes:

- **Raw ranking visualisation (Flourish)**  
- **Trend analysis chart (Matplotlib)**  
- **Per-capita comparison (Matplotlib)**  
- **Country profiling scatter plot (Matplotlib)**  
- Final presentation summarising findings
